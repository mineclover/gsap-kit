🧩 Test Runner Component Demo

Web Component로 간단하게 테스트 러너 추가하기

📖 사용법

HTML에 단 한 줄만 추가하면 테스트 러너가 자동으로 생성됩니다:

<test-runner spec-file="/test-specs/gsap.spec.json"></test-runner>

속성 (Attributes)

  • spec-file: 테스트 스펙 JSON 파일 경로 (필수)
  • visualize: 마우스 경로 시각화 여부 (선택, 기본값: true)
  • slow-motion: 슬로우 모션 모드 (선택, 기본값: false)
